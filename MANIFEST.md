# MANIFEST.md

## Purpose

This repository publishes Owlbox, a repository-local continuity system.

## Contents

- `README.md`: generated landing page.
- `PROJECT.md`: project charter.
- `SOP.md`: procedure and operating specification.
- `FILES.md`: file descriptions.
- `TEMPLATES.md`: template documentation.
- `BUILDABOX.md`: build documentation.
- `VISION.md`: generated concatenation of Owlbox project files.
- `OWLBOX.md`: generated continuity view for this repository.
- `owlbox/`: canonical continuity sources for this repository.
- `templates/`: reusable Owlbox source templates.
- `codex/owlbox/`: Codex skill implementation.
- `scripts/`: build scripts.

## Rules

- Do not confuse `VISION.md` with `OWLBOX.md`.
- Do not make original edits directly in generated files.
- Keep the Owlbox deliverable separate from repository support files.
